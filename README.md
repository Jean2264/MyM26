#SJ
Este proyecto es un sistema de gestion de venta, diseñado para administrar ventas con punto de venta, clientes, empleados, proveedores,usuarios con sus permisos, ver resumenes de ventas con sus detalles, filtrados personalizados, un modulo en de se puede ver el estado menual, y anual de las ventas, ver movimientos en la app, manejar entradas y salidas, el proyecto esta diseñado y desarrollado en Winforms C#, con base de datos en SQL Server.

Tambien a medida que voy aprendiendo habilidades nuevos voy sumando cosas nuevas a la app como tambien, escalabilizar el codigo, en paralelo como estoy aprendiendo sobre API en ASP.NET Core, en donde ya cuento con conocimientos en EndPoint, metodos HTTP, autenticacion, hasheos de contraseñas para mas seguridad como por ejemplo con BCrypt o AES.
